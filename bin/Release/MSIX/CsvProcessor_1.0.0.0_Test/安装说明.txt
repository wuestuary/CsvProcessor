CsvProcessor 安装说明
=====================

文件清单
--------
CsvProcessor_1.0.0.0_x64.msix 	Windows 应用程序安装包（主文件）
CsvProcessor_1.0.0.0_x64.cer		数字证书（安装前需信任）
Install.ps1					自动安装脚本（推荐）
安装说明.txt 					本文档


安装方法（推荐）
================

方法一：自动安装（最简单）
-------------------------
1. 右键 Install.ps1 → "使用 PowerShell 运行"
2. 在弹出的 UAC 提示中点击"是"（允许管理员权限）
3. 等待安装完成，按任意键退出

脚本会自动安装证书和应用，无需手动操作。


方法二：手动安装
----------------

步骤 1：安装数字证书
1. 双击 CsvPublisher.cer
2. 点击"安装证书"
3. 选择"本地计算机" → "下一步"
4. 选择"将所有的证书都放入下列存储"
5. 点击"浏览" → 选择"受信任的根证书颁发机构"
6. 点击"确定" → "下一步" → "完成"

步骤 2：安装应用
1. 双击 CsvProcessor_1.0.0.0_x64.msix
2. 点击"安装"
3. 等待安装完成

如果提示"无法验证此应用包的发布者证书"，请确认步骤 1 的证书已正确安装。


常见问题
========

问题 1：提示"无法验证此应用包的发布者证书"
原因：证书未安装或安装位置不正确
解决：
- 重新运行方法一的自动安装脚本，或
- 手动安装证书到"受信任的根证书颁发机构"（不是"个人"）

问题 2：提示"Windows 已保护你的电脑"
解决：
1. 点击提示框下方的"更多信息"
2. 点击"仍要运行"

问题 3：右键没有 PowerShell 选项
解决：
1. 按 Win + X → 选择"终端（管理员）"或"PowerShell（管理员）"
2. 输入：cd 安装包所在文件夹路径
3. 输入：.\Install.ps1
4. 按回车运行


使用应用
========

打开方式
- 开始菜单：搜索 "CsvProcessor"
- 桌面快捷方式：安装后自动创建（如有）
- 文件关联：右键 .csv 文件 → "打开方式" → "CsvProcessor"

文件关联设置
1. 右键任意 .csv 文件
2. 选择"打开方式" → "选择其他应用"
3. 选择"CsvProcessor"
4. 勾选"始终使用此应用打开 .csv 文件"


卸载方法
========
1. 打开 设置 → 应用 → 已安装的应用
2. 找到 "CsvProcessor"
3. 点击 "..." → "卸载"


系统要求
========
操作系统    Windows 10 版本 1809 或更高 / Windows 11
架构        x64（64位）
运行时      自动包含（无需单独安装 .NET）


技术支持
========
如有问题，请检查：
1. 系统版本是否满足要求
2. 证书是否正确安装
3. 是否以管理员权限运行安装脚本


建议：日常使用选择方法一（自动安装），简单快捷且自动处理证书问题。
